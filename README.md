## Overview

Egators is a responsive educational platform UI built with vanilla HTML, CSS, and JavaScript. It spans four pages with smooth navigation, interactive components, and a mobile-first responsive layout — covering everything from course discovery to team profiles and direct contact.

---

## Pages

| Page | Sections |
|:---|:---|
| Home (`index.html`) | Hero, Categories, Popular Courses, FAQ Accordion, Testimonials Slider, Footer |
| About (`about.html`) | Achievements Stats, Meet the Team, Footer |
| Courses (`course.html`) | Full Course Catalog (9 courses), Footer |
| Contact (`contact.html`) | Contact Aside Panel, Contact Form, Footer |

---

## Features

| Feature | Description |
|:---|:---|
| Responsive Layout | Adapts across desktop (1024px+), tablet, and mobile (600px) |
| Fixed Navbar | Scroll-triggered background color change |
| Mobile Menu | Animated slide-in navigation with open/close toggle |
| FAQ Accordion | Click to expand/collapse answers with icon toggle |
| Testimonials Slider | Swiper.js carousel with pagination dots |
| Team Hover Effects | Grayscale-to-color image + slide-in social links |
| Contact Form | First name, last name, email, message with submit button |
| Footer | Links, privacy pages, contact info, social icons |

## Tech Stack

| Tool | Purpose |
|:---|:---|

## How to Run

**1. Clone the repository**
```bash
git clone https://github.com/your-username/egators.git
cd egators
```

**2. Open in browser**

No build tools or dependencies needed. Simply open `index.html` in your browser.

```bash
# Or with VS Code Live Server
code .
```

**3. Navigate pages**

Use the navbar links to move between Home, About, Courses, and Contact.
